1- No contexto da computação, um "arquivo" refere-se a uma unidade de armazenamento de dados identificada por um nome. Pode conter informações, como texto, imagens, programas, ou qualquer tipo de dados. Os arquivos são organizados em sistemas de arquivos e são manipulados por programas para leitura, gravação e manipulação de dados.
1.1- Um arquivo de texto é um tipo de arquivo que armazena dados em formato de texto simples, geralmente codificado em caracteres ASCII ou Unicode. Este formato é composto por linhas de texto, onde cada linha representa uma sequência de caracteres. Arquivos de texto podem conter informações legíveis por humanos e são comumente usados para armazenar código-fonte, configuraçõs, documentos simples e dados tabulares. Ao contrário de formatos binários, os arquivos de texto podem ser facilmente lidos e editados com editores de texto simples.
2 - Em programação, uma variável é um espaço de armazenamento identificado por um nome, capaz de conter informações que podem ser modificadas durante a execução do programa. Cada variável possui um tipo de dado associado, determinando o tipo de valor que pode armazenar, como números ou caracteres. As variáveis são essenciais para a manipulação de dados em um programa, permitindo o armazenamento, cálculos e controle do fluxo de execução com base nos valores que contêm. Elas são referenciadas por identificadores e podem ser atribuídas com novos valores ao longo do código.
3 - Um servidor, no contexto da computação, é um sistema de computador ou software projetado para fornecer serviços, recursos ou dados para outros dispositivos chamados clientes em uma rede. Esses servidores atendem a solicitações dos clientes, que podem envolver o fornecimento de dados, execução de serviços específicos ou processamento de informações. Conectados em redes, como a internet, eles são especializados para diferentes finalidades, como servidores web, de banco de dados, de e-mail, entre outros. Operando continuamente, os servidores desempenham um papel crucial na arquitetura de redes, garantindo a disponibilidade constante dos serviços que oferecem.
