1 - Explique as diferenças entre 3 linguagens de programação
R: 3 linguagens de programação são: Python, PHP e C++
Python: Versátil e usado em várias áreas, incluindo desenvolvimento web, automação, ciência de dados e aprendizado de máquina; Suporta programação orientada a objetos, funcional e procedural; Gerenciamento automático de memória com coleta de lixo; Sintaxe simples e legível; Grande comunidade, ativa em diversas áreas, com uma variedade de bibliotecas e frameworks; Desenvolvimento web, automação, aprendizado de máquina, ciência de dados.

PHP: Projetado especificamente para desenvolvimento web do lado do servidor, com foco em criar páginas dinâmicas; Principalmente usado de forma procedural, mas suporta programação orientada a objetos; Gerenciamento automático de memória; Sintaxe orientada a scripts com foco em desenvolvimento web; Comunidade forte no desenvolvimento web, com suporte extensivo em servidores web e frameworks; Desenvolvimento web do lado do servidor, criação de páginas dinâmicas.

C++: Linguagem de programação geral, com ênfase em desempenho, utilizada em sistemas embarcados, jogos e desenvolvimento de software de sistema; Suporta programação orientada a objetos e procedural, com destaque para desempenho; Requer gerenciamento manual de memória; Sintaxe mais complexa, especialmente em relação a manipulação de ponteiros e gerenciamento de memória; Comunidade robusta, especialmente em jogos, sistemas de baixo nível e desenvolvimento de software de sistema; Jogos, sistemas embarcados, software de sistema, aplicações de desempenho crítico.
